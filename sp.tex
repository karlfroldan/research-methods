\begin{execsummary}
	The A* algorithm is one of the most popular branch and bounding algorithms for solving combinatorial problems such 
    as pathfinding. This paper aims to find an efficient implementation of the A* algorithm on a parallel computing 
    environment using a functional programming language. Specifically, the algorithms experimented on are 
    HDA* and PNBA* star, two distinct parallel A* algorithms. The proposed project hopes to combine the 
    efficiency of the A* algorithm and the correctness of functional programming.
\end{execsummary}
